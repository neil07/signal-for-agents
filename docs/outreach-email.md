## Sending Tips

- **Best send window:** Tue / Wed, 7:30–8:30 AM local time (agents check email before first showing)
- **Personalization is non-negotiable:** Always fill `{{City}}` and `{{First Name}}` with real data — generic = spam folder
- **A/B test subject line:** Backup option → `"Your next listing post — done before you park the car"`
- **Track:** Monitor Open Rate and CTR. Low opens → tweak subject. Low clicks → shorten body further
- **Follow-up:** If no reply after 3 days, send a short nudge (2–3 lines max, same WhatsApp CTA, no new info)
